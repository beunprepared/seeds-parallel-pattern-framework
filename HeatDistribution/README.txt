This project shows how to use the Stencil pattern in Seeds.

This project also makes use of the pattern adder operator to add termination detection to the program.

To run this project:
	The simplest way to view and execute this program is to load the source using Eclipse.

	The project should have the project PGAFramework in its dependencies.

	The libraries listed in the Seeds shuttle folder should also be added to the list of dependencies.

	Change the path in Seeds.start( "") to point to the shuttle folder in the local computers.
	
	Execute the class RunHeatDistribution
